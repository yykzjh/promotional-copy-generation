## 核心约束

- **全部本地部署**：使用 vLLM 等本地部署主 LLM 及其他 LLM；Agent 直接调用；图片生成由多模态 LLM 处理，无需单独集成
- **合规校验**：输入（需求、描述、图片）与输出（文案、图片提示词、生成图片）均须通过安全校验
- **模块化可扩展**：Skills、阶段上下文、MCP Servers 均采用配置/注册表驱动，便于扩展

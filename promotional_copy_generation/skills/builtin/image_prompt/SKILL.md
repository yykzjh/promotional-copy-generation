# Image Prompt Guidelines

## Structure
- Subject description
- Style/atmosphere
- Composition suggestions
- Lighting and color

## Style Keywords
- Commercial photography, product shots, atmospheric, premium feel
- Choose based on promotional scenario

## Negative Prompt
- ugly, blurry, low quality
- Adjust based on specific needs
